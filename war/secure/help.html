Project Bot Help

  

Add the Status Bot to your Slack Team

From the "Team to Work With" drop down, select "Create new team..."

Then select your team from the list. Click the Refresh icon if your team doesn't show up.

Create an Incoming Webhook

This allows messages to be posted directly to the team, such as the daily status updates

You'll need to be a project admin. Open the App Manage page on Slack in a browser for your team: https://[MYTEAMNAME].slack.com/apps/manage

Choose the Custom Integrations tab and then Incoming Webhooks

Add a new configuration

Make a copy of the Webhook URL value to store in the configuration. Paste the URL and click Save URL

Add a Slash command

The Slackbot relies on /slash commands rather than the Realtime Messaging API from Slack

To configure a slash command, select "Build your own" in the upper right hand corner of the page

And then select "Make a custom integration"

Select Slash Commands from the list

Choose the command to summon the bot. For consistency, we're using /statusbot

The URL to call is https://slackbot-1225.appspot.com/bot/slash

Paste the Token into Verfication Token and click Save Key

Save your new integration on Slack and you're ready to add dates to the bot.
